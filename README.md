## 🇩🇪 Instant Kill Hand Tool (Vernichtungswerkzeug)
Ein hocheffizientes Mod-Werkzeug für Factorio 2.1+ / Space Age, mit dem du lästige Alien-Plagen im Vorbeigehen pulverisieren kannst.
## ✨ Features

* Sofortige Vernichtung: Eliminiert alle Beißer, Spucker, Würmer und Nester im markierten Bereich.
* Effektvolle Explosionen: Feindliche Basen und Würmer explodieren spektakulär beim Aufprall.
* Komfortable Bedienung: Lässt sich bequem per Shortcut-Leiste (Bildschirmrand) oder über die Tastenkombination SHIFT + K aufrufen.
* Sauberes Interface: Das Werkzeug blockiert keinen Platz in deinem normalen Inventar oder Handwerksmenü.
* Space Age Bereit: Funktioniert universell auf Nauvis sowie allen neuen DLC-Planeten (wie Gleba, Vulcanus etc.).

## 🚀 Installation (Manuell)

   1. Lade das Repository als ZIP-Datei herunter.
   2. Benenne den Ordner exakt in InstantKillTool_1.0.0 um (falls nötig).
   3. Verschiebe den Ordner in dein Factorio-Mod-Verzeichnis:
   * Windows: %APPDATA%\Factorio\mods\
      * Linux: ~/.factorio/mods/
      * macOS: ~/Library/Application Support/factorio/mods/
   4. Starte Factorio, aktiviere die Mod im Menü und aktiviere den Button in der Shortcut-Leiste über das Drei-Punkte-Menü (...).

## 🇬🇧 Instant Kill Hand Tool
A highly efficient modding tool for Factorio 2.1+ / Space Age designed to instantly vaporize annoying alien infestations.
## ✨ Features

* Instant Demolition: Eliminates all biters, spitters, worms, and spawners inside the selected area.
* Impact Explosions: Enemy structures and turrets blow up with a spectacular visual effect.
* Easy Access: Toggle the tool instantly via the shortcut bar icon or by pressing SHIFT + K.
* Clean User Interface: The item is completely hidden from your crafting menu and does not clutter your inventory.
* Space Age Ready: Works seamlessly on Nauvis as well as on all new DLC expansion planets (Gleba, Vulcanus, etc.).

## 🚀 Installation (Manual)

   1. Download the repository as a ZIP archive.
   2. Rename the folder to exactly InstantKillTool_1.0.0 (if necessary).
   3. Move the directory into your Factorio mods folder:
   * Windows: %APPDATA%\Factorio\mods\
      * Linux: ~/.factorio/mods/
      * macOS: ~/Library/Application Support/factorio/mods/
   4. Launch Factorio, enable the mod, and activate the button in your shortcut bar using the three-dots menu (...).

## 🛠️ Technical Details / Technische Details
This mod bypasses the strict Factorio 2.1 engine entity-filtering rules for custom selection tools by reading the raw coordinates (event.area) and scanning the surface manually using a custom find_entities_filtered routine.
Diese Mod umgeht die strengen Filter-Regeln der Factorio 2.1 Engine, indem sie die rohen Koordinaten des Rahmens (event.area) abfängt und den Bereich manuell per find_entities_filtered nach feindlichen Objekten absucht.
